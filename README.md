# Map
* 美观、易用的用户界面，保证用户直接上手便可以使用。
* 使用GPS获取当前位置，获取当前位置的天气详细信息与四天气预报。
* 详细天气实况信息，包括各实时温度、风向、湿度。
* 地图功能包含实时路况、普通地图、导航地图、夜景地图、卫星地图、连续定位、一次定位、导航等功能。
